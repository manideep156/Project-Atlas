# 📚 Topics Covered

## Introduction to Pandas

Learned:

- What Pandas is
- Why Pandas exists
- Relationship between NumPy and Pandas
- Real-world applications of Pandas

### Key Point

NumPy is optimized for numerical computation, whereas Pandas is designed for working with structured and labeled datasets.

---

## Pandas Series

Covered:

- Creating Series from Lists
- Creating Series from NumPy Arrays
- Creating Series from Dictionaries
- Custom Indexes
- Accessing Values
- Accessing Indexes
- Data Types

### Key Learning

A Series behaves similarly to a NumPy array but includes meaningful labels for every value, making data easier to organize and access.

---

## Pandas DataFrames

Covered:

- Creating DataFrames
- Rows and Columns
- Column Labels
- Index
- Mixed Data Types

### Key Learning

A DataFrame is the primary data structure in Pandas and resembles an Excel spreadsheet or SQL table.

# 🧠 Key Learnings

- Pandas extends NumPy rather than replacing it.
- Labels make working with structured data significantly easier.
- DataFrames are the foundation of most data analysis workflows.
- Data inspection should always be the first step before analysis.
- Filtering and selecting data become much more intuitive using Pandas.
