# 🛍️ Retail Store Sales Analytics using NumPy

## 📌 Project Overview

This project analyzes weekly sales data for multiple retail products using **NumPy**.

The objective is to perform data exploration, statistical analysis, product performance evaluation, weekly trend analysis, business simulations, and generate actionable insights using only NumPy.

This project is part of **Project Atlas**, my structured learning journey in Data Analytics and Artificial Intelligence / Machine Learning.

---

# 🎯 Objectives

- Explore a real-world style dataset
- Perform descriptive statistical analysis
- Compare product performance
- Analyze weekly sales trends
- Apply Boolean filtering
- Rank products and weeks
- Simulate business scenarios using broadcasting
- Derive business insights from numerical data
